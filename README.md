## ১। undefiend এবং null এর পার্থক্য

undefiend হলো একটা ভেরিয়েবল ডিক্লিয়ার করার পরে যখন কোনো ভ্যালো/মান দেওয়া হয় না বা ভেরিয়েবল কোনো ভ্যালো/মান খোঁজে পায় না তখন undefiend হয়।  
null হলো একটা ভেরিয়েবল ডিক্লিয়ার করার পরে Developer নিজে থেকে কোনো কারণে খালি রাখে বা null ভ্যালো দিয়ে থাকে।

## ২। map এবং forEach এর পার্থক্য

map এবং forEach এর বড় পার্থক্য হলো map নতুন একটা array রিট্রান করে কিন্তু forEach কিছু রিট্রান করে না।  
map সব গুলা এলিম্যান্টকে ধরে নতুন array তৈরি করে।  
forEach একটা array এর মধ্যে লুপ চালিয়ে তাকে বলে দেয়া এ্যাকশন গুলা সম্পন্য করে।

## ৩। == এবং === এর পার্থক্য

== শুধু মাত্র মান চেক করে এটা কোন ধরেনের ডাটা তা চেক করে না কিন্তু === মান চেক করার সাথে ডাটা টাইপ চেক করে।

## ৪। async/await

async/await হলো মর্ডান জাভাস্ক্রিপ্টেত ডাটা হ্যান্ডেলিং এর একটি মাধ্যম।  
যার ধারা ডাটা ফেচিং করা যায় আরো সহজ ভাবে এবং এটি স্টেপ বাই স্টেপ আগায়।  
একটি ডাটা পাওয়া অব্দি অপেক্ষা করে যার জন্যে এরর বুঝা এবং এরর হ্যান্ডেল করা আরো সহজ হয়েছে।

## ৫। Scope in JavaScript

Scope হলো ডাটা/ভেরিয়েবল কে একটি নির্দিষ্ট যায়গার ভিতরে সেট করে দেয়া।  
যে এই ডাটা/ভেরিয়েবল শুধু এই এই যায়গায় ব্যবহার করা যাবে।

Scope ৩ ধরণের Global, Functional, Block  

Global হলো এটি যেকোনো যায়গা থেকে এক্সেস করা যাবে।  
Functional হলো এটি শুধু মাত্র Functional এর ভিতরে এক্সেস করা যাবে।  
Block হলো সাধারনত একটা এরিয়া ডিক্লিয়ার করা হয় {} এর মাধ্যমে যার ভেরিয়েবল let, const দিয়ে ডিক্লিয়ার করা হয় এবং এই ভেরিয়েবল গুলো শুধু মাত্র {} এর ভিতরেই এক্সেস করা যায়।
